# Proyecto-Final-BDA

Montar Proyecto a GCP
1. Subir el archivo proyecto.zip a la instancia de gcp
2. Descomprimir el archivo Proyecto.zip

Dar permisos para ejecutar
- Ingresar a la carpeta Proyecto: cd Proyecto
- Dar permisos de ejecución al proyecto: chmod +x Database.sh

Ejecución del bash:
1. sudo ./Database.sh
2. Escribir Contraseña: 666
